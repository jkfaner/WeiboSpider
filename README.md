# WeiboSpider

> 微博爬虫系统，高效率博主图片视频爬取。数据接口均来自weibo.com

[![GitHub stars](https://img.shields.io/github/stars/jkfaner/apple-monitor.svg)](https://github.com/jkfaner/apple-monitor)

### 功能

爬取关注用户的图片数据，包括但不限于视频、livephoto数据。

### 特点
- 设置灵活
- 数据精准
- 稳定效率高

### 新增功能
- 自定义登录方式
- 自定义爬虫模式
- 自定义只爬用户
- 自定义排除用户
- 自定义数据筛选规则
- 监听模式

### 使用说明
1. 环境支持
```
- python 3 # 脚本执行环境
- mysql # 账户数据储存
- redis # 缓存
```
2. 依赖安装
3. 配置文件设置`config.ini`
4. 手动配置爬取用户
